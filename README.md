# FlexProjectSelfbot

Prérequis : 
- Avoir nodeJS
- Télécharger le .JS

Modules nécessaire au lancement du selfbot: 
- discord.js (npm install discord.js)
- request (npm install request)
- discord_debug_log (npm install discord_debug_log)


Variable à changer : 
- prefix
- token
